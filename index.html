Projects

Network Design with Two Switches and VLAN Implementation

In this project, a network was designed using two switches connected via a trunk link. End devices were distributed across both switches and assigned to different VLANs based on departmental segmentation. The aim was to improve network performance and security through logical separation. Trunk ports were configured to allow inter-VLAN communication between switches.

VLAN Configuration, Access and Trunk Port Setup, Switch-to-Switch Trunking, Network Segmentation

Network Design Using Three Routers with Static Routing

This project involves designing a network topology with three routers connected through serial interfaces to simulate WAN communication. Each router connects to a separate LAN, and static routing was configured to enable inter-network communication. This setup reflects real-world branch connectivity across distant locations.

Static Routing Configuration, Serial Interface Setup, Subnetting, WAN Design Concepts

VLAN Implementation with Inter-VLAN Routing

This project features a comprehensive VLAN setup on a LAN network. End devices were segmented by departments using VLANs. A router-on-a-stick method was implemented for inter-VLAN routing, allowing communication between different VLANs through a single router interface. This project demonstrates practical application of VLANs in enterprise environments.

VLAN and Inter-VLAN Routing, Access and Trunk Port Configuration, Router-on-a-Stick Implementation, Network Testing (Ping, CLI Commands)
